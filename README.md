# 项目简介

## 核心功能
- 基于YOLOv8的坐姿检测系统，支持4类姿态识别（良好坐姿、后仰、前倾、驼背）
- 提供918张标注数据集，包含完整训练集和测试集
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整UI界面，可实时显示检测结果、目标位置、目标总数及置信度
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持批量检测并查看所有结果
- 提供模型选择功能，可根据需求切换不同模型
- 支持检测结果保存功能

## 适用场景
- 办公室坐姿监测与提醒系统
- 学校教室学生坐姿管理
- 健康管理与姿势矫正应用
- 人机交互研究中的姿态识别
- 智能家居中的坐姿监测功能

## 技术特点
- 使用PyCharm/VSCode开发环境，基于Python 3.9
- 依赖opencv-python、PyQt5、ultralytics、torch1.9等开源库
- 采用YOLOv8作为核心检测算法
- 提供完整的源代码、UI界面源文件、数据集及运行教程
- 已训练好可直接使用的模型，配置环境后即可运行
